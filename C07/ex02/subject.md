<p>• Create a function ft_ultimate_range which allocates and assigns an array of ints.
This int array should contain all values between min and max..</p>
<p>• Min included - max excluded.</p>
<p>• Here’s how it should be prototyped :</p>

```C
int ft_ultimate_range(int **range, int min, int max);
```

<p>• The size of range should be returned (or -1 on error).</p>
<p>• If the value of min is greater or equal to max’s value, range will point on NULL
and it should return 0.</p>
